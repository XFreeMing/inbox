## 如何通过openclaw构建agent大军
## How to Build an Agent Army with OpenCLAW

- I want to build a subagent that you will use for all coding activities. anytime you nees to code something ,you will use this subagent. Please name the subagent jason and have it run on KIMI. Let me know what we need to do to set up that oauth or API key.


中文翻译: 我想创建一个子代理，你会在所有编程活动中使用它。任何时候你需要编写代码时，都会使用这个子代理。请把这个子代理命名为jason，让它运行在KIMI上。告诉我需要做什么来设置那个oauth或API密钥。

单词/词组解释:
- subagent - n. 子代理（帮助执行任务的AI助手）
- coding activities - 编程活动
- anytime - adv. 任何时候
- needs - v. 需要
- name - v. 命名，给...取名
- run on - 在...上运行
- set up - v. 设置，建立
- oauth - n. 开放授权（一种网络认证方式）
- API key - n. API密钥（访问接口的密码）
